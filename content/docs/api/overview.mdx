## Introduction

The MrScraper API provides a powerful and flexible way to programmatically scrape web pages and retrieve structured data. Built on REST principles, our API offers predictable resource-oriented URLs, accepts JSON-encoded request bodies, returns JSON-encoded responses, and uses standard HTTP response codes.

## Base URL

All API requests should be made to:

```
https://api.v3.app.mrscraper.com
```

## API Features

<Cards>
  <Card title="JSON Format" icon={<Braces />}>
    All requests with a payload require `Content-Type: application/json` header. All responses are returned in JSON format.
  </Card>
  <Card title="Authentication" icon={<Key />}>
    All API endpoints require authentication using an API key passed in the `x-api-token` header. See the [Authentication](/docs/api/authentication) page for details.
  </Card>
  <Card title="Pagination" icon={<ListOrdered />}>
    List endpoints support pagination to handle large datasets efficiently. See the [Pagination](/docs/api/pagination) page for details.
  </Card>
  <Card title="Error Handling" icon={<CircleAlert />}>
    The API uses standard HTTP status codes and returns detailed error messages. See the [Error Handling](/docs/api/error-handling) page for details.
  </Card>
</Cards>
